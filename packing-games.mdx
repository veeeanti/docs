# Packing games UC-style

The team uses 7-Zip for all uploads. Multi-part uploads are no longer required, but the split convention below still applies.

## Steps

1. Rename the main game folder to `{Full game name} - UC`.
2. Drop a `UnionCrax.url` file pointing to the site into the folder root.
3. Confirm the folder contains all DLCs and no extras like artbooks or soundtracks.
4. Create a 7-Zip archive of the folder.
5. Set split volumes to `10240m` (~10 GB).
6. Set archive format to `.7z`.

That's the full packing process.
