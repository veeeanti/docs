# Cracking games

While this isn't cracking in the strict sense, the team calls it that. This page covers simple DRM bypassing only. **Denuvo and any DRM other than Steamworks are out of scope** — the team does not crack those.

Skip this entire page if the game is already DRM-free (for example, from GOG).

## Apply cracks

Apply cracks and fixes directly to the game folder. If an Online fix requires extra dependencies (such as Steam), put them in an `_Online-fix` folder at the game root.

## Goldberg Steam Emu

Goldberg Steam Emu is a Steam API replacement that adds offline functionality. The team uses [`gbe_fork`](/tools-we-use#gbe_fork), a maintained fork, by default.

## Steamless

When replacing the Steam API isn't enough and the game still errors out due to SteamStub, use Steamless to unpack the executable.

1. Download Steamless.
2. Run `Steamless.exe`.
3. Select the real game executable.
4. Click **Unpack file**.
5. Delete the original executable.
6. Rename the unpacked file to the original executable name.

## Online-fix

Online-fix.me publishes multiplayer fixes for many games.

1. Create an account on online-fix.me.
2. Search the game and open the matching result.
3. Download the fix from the **Fix Repair** folder.
4. Extract it (password: `online-fix.me`).
5. Move the fix into the `_Online-fix` directory at the game root.

## DRM cracks

For everything else, look for crack-only files on cs.rin.ru and apply them to your own clean game files.
