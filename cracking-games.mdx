# Cracking Games

While this isn't cracking in a true sense, it's called cracking. This part will only cover simple DRM bypassing. Denuvo, or any other form of DRM than steamworks will not be cracked by us.

This step can be completely skipped if the game is DRM-free.

## Apply Cracks

Apply all cracks/fixes directly. However if an Online fix requires additional dependencies (such as Steam) put them in a `_Online-fix` folder in the game root.

## Goldberg Steam Emu

Goldberg Steam Emu is a Steam API replacement that adds offline functionality.

## Steamless

Sometimes just replacing the steam API isn't enough, and running the game still results in an error due to steamstub.

1. Download steamless
2. Run Steamless.exe
3. Select the real game exe
4. Click unpack file
5. Delete the original executable
6. Rename the unpacked exe to the original name

## Online-fix

1. Make an account on online-fix.me
2. Search the game, click the correct result
3. Download the fix from the Fix Repair folder
4. Extract (password: online-fix.me)
5. Put the fix into `_Online-fix` directory

## DRM Cracks

We usually find crack-only files on cs.rin.ru and apply them to our own game files.
