# Downloading Games to Upload

We prefer to use CSF on cs.rin.ru uploaded by its members, or by using depotdownloader or Accela, or by just buying the games, or if we already have it in our library we'll use it.

We do not reupload other sites' releases, that's cheap and not what we do. Fitgirl, Myabandonware, crack releases, releases of the sort are exceptions.

## Sources

- **cs.rin.ru** - Clean Steam Files from community members
- **Depot Downloader** - Download from Steam using app/depot/manifest IDs
- **Accela** - Download tool with daily generation limits
- **GOG** - DRM-free downloads
- **ROMHEAVEN** - ROMs and CSF

## Important

Always check cs.rin.ru thoroughly for the game before assuming it doesn't exist.
