Getting CSF (Clean Steam Files)
There are mainly 3 sources we use to get game files: Ripping directly from Steam using an account which has the game purchased, searching cs.rin forums, or using other platforms, that don't include DRM in their games (GOG babyyyyy).
GOG is by far the main one reccomended, because no DRM.

Steam
For this approach you will obviously need a Steam account, with the game you want to download purchased.
We will use Depot Downloader GUI, but if you preffer other depot downloader, feel free to use that one instead.

First visit SteamDB, and search up the game you want to download.
Then copy the number, that corresponds to the games app id, and paste it into DDGUI like below.
Next go to the depot tab, select the windows depot, copy the corresponding depot id, and paste it like below. Here also take note of the build id.
Finally, go to the manifests tab, copy the most recent one, and paste again into the app like shown below.
Now in the app just enter your steam login details, select where to download the game, and press download.
DDGUI set up

Some games however have more than just one depot for each platform. Let's take Tomb Raider GOTY Edition as an example:
Tomb Raider GOTY Edition Depot list
Here it will be a bit more complex.
First of all, ignore all depots, that are not tagged Windows. We are only listing games for Windows, so there is no reason to download other platforms.
Next, don't download any other languages, other than english. Most people are english speaking, and don't care about the rest. So it only bloats the download by a lot.
Next up, the logic of the game is split into multiuple depots here: Main content, and Executable. Download both. The game won't be complete if one is not present.
Last of all are DLC depots. Unless all of them are accessible from your account, don't even bother uploading those. If they are, check that all of them are actually real DLCs you can purchase, and download each one with an exception of optional files.
Don't download any redist depots, as they are just software needed to be installed in order to run the game.
In my example this leaves us with the following depots: Main content, Executable, English, and the 5 depots starting with DLC.
So to recap: Download all depots, tagged Windows, Don't download any other language, other than English, download all DLCs, and don't download redists, debug files, or optional depots.

Once the files have been downloaded, in each depot delete the .DepotDownloader folder.

At this point, you should have a folder with the steam app id on it with CSF in it.

cs.rin
For this approach you will need an account on the forums. You can do this by visiting the website and registering.
With an account ready, let's download the game. Start by clicking the search button or going to this link. There enter the name of the game you want to download and hit enter.
From the results that show up, find the thread, that is named after your game. If there are more threads with the same name, click on the one labeled as [info].
Once in the thread, go to the last page, work your way back the thread, and find a download link, that works. The reason for this is to find the newest build aviable. Then just download the game files.
If during any process anything will ask for a password, it is most likely cs.rin.ru, unless the uploader specifies otherwise.
Important thing to do is to make a note of what build id the files are from. Build id indicates version of the game, which we will later need.

GOG
GOG is a special kind of game store, because the games are DRM-free (which is the best kind of DRM). GOG should always be checked first, because no cracking is necessary.
This one is the easiest option. All you have to do is visit this link, search for the game you want, and then download the files.
It sadly doesn't end here, because the files inside the archive you just downloaded is a setup. so after extracting it, and installing the game, go to the install location, and copy all of the game files into a separate folder.
Once again take note of the game version written in the name of the setup for later use. The version number is the one in front of the number in brackets, not in the brackets.
Once everything is copied, you can uninstall the game. One more step. GOG includes a bunch of files in the game folder, that aren't needed to run the game. We will need to delete them. Do so as shown below.

Cracking the game
While this isn't cracking in a true sense, I don't know how else to call it, so cracking it is.
This part will only cover simple DRM bypassing. Denuvo, or any other form of DRM than steamworks will not be cracked by us.
This step can be completely skipped if the game is DRM-free.

Apply all cracks/fixes directly. However if an Online fix requires additional dependencies (such as Steam) put them in a _Online-fix folder in the game root one.
This means that if a person would copy all contents of the _Online-fix folder into the game root, the multiplayer crack would be applied and worked.
However if it is a multiplayer game and such fix is simply required, apply it directly.

Goldberg
Goldberg Steam Emu is a Steam api, that can be used as a replacement for the original one to add offline functionality (in other words not require owning the game or connecting to steam).

Learn how to use it here.

Steamless
Sometimes just replacing the steam api isn't enough, and running the game still results in an error. The cause is probably steamstub, Steam's other form of DRM.
Luckily for us, it is just as easy to remove.

First download steamless from here. In the downloaded archive run Steamless.exe. It will look like this:
Steamless UI
The only thing you need to do is click on the three dots in the first option, find and select the real game exe (be aware, that some games, mainly on unreal engine, have their main executable nested in the folder hiearchy), and click unpack file.
After this is done, delete the original executable and rename the new one (ending in unpacked.exe) to the name of the original one.

Online-fix
Online fix is a site, that makes cracks for games to allow multiplayer on them.
You will need to make an account here, and maybe even install autotranslating extension.

Getting the crack is pretty straight-forward. You search the game, click on the correct result, click on download the fix from the server, and download it from the Fix Repair folder.
After the fix has finished downloading, extract it. The password for the archive is online-fix.me.
Now put the fix into _Online-fix directory.

Correctly packaging the game
This is the most critical step. Here we will make the first thing an user will see after downloading.

First of, the folder contents. Take the root of your game files, and put them in a new folder. In most games, their executable will be here.
The files should include all DLCs, and no extras like artbooks, or soundtracks.

Into this folder put an url file (download here and rename to UnionCrax.url) linking to our site.

Now rename the root game folder. the format is following: {Full game name} - UC. For example Geometry Dash - UC.

The final step is to create an archive to upload. To do this install 7-Zip.
With 7-Zip, select the folder we just named, and click Add to archive.
On the menu, that pops up, ensure the archive is called after the game folder, and its format is .7z.
Last thing we want to change before compression is the part size. Due to the best setup, we need to support hosts that limit uploads to smaller sizes. Thus we need to split the archive. Into the section called Split to volumes enter 10240m, which is equivalent to roughly 10 GB.
Now just wait till 7-zip is done compressing.
This is the final file we will upload.

Uploading
For uploading, we use our own mirror tool. For info in installing the tool you should see our discord team server.

Listing the game
After you are done uploading your game with our tool, we need to list it.
The only thing you have to do, is copy the output json of the tool and if you have access to the game mangement page you need to fill the fields in.
Along with the file links, also include the version as labeled on the game upload, the source of your crack and size of the unarchived game folder.
Crack source means that if you used goldberg, say you used goldberg. If you used GOG instead of Steam say you used GOG. If you got it from a scene release or a repacker, then you check what scene group cracked the game, and list them.
Then you just click add/update game and you are done, our tool also has a link to game (auto push links) feature, (currently no support for new game versions) so you dont have to manually copy over the output json.
