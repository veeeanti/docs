# Tools We Use

You may have noticed we often use "gbe_fork" in our uploads. This is a steam emu - and arguably one of the better ones out there. It is a fork of Goldberg Steam Emu maintained by detanup01 on Github. This is the most up to date emu in terms of steam updates and interfaces.

## gbe_fork

A fork of Goldberg Steam Emu maintained by detanup01. Recommended for steam game emulation.

## Online-fix.me

We use online-fix.me or our own uc-online / uc-online2 for multiplayer fixes. Sometimes we'll use Unsteam, sometimes we'll get a game that works online even with gbe_fork applied.

## DRM Cracks

We usually find crack-only files on cs.rin.ru and apply them to our own game files.

## Right-click Script

There is a script on GitHub that you can add to your right-click menu to automatically apply gbe_fork to steam games. It generates interfaces.txt automatically as well as a URL file linking to the site. It also creates a shortcut batch script.

The script has both a Linux version and standard Windows version.

## Accela

We use Accela mostly to download games too, since donations were made to morrenus, there are 55 daily generations to use with Accela.

## File Mirrorer

Our file mirrorer is broken currently.

## Denuvo

We do not deal with demo bypasses with Denuvo, and we are very on-the-fence with hypervisor stuff from DenuvOwO, just because of the risks that our users will have to take.
