# Tools we use

This page lists the tools the team relies on for sourcing, cracking, and packing games.

## gbe_fork

A fork of Goldberg Steam Emu maintained by [detanup01](https://github.com/Detanup01/gbe_fork) on GitHub. It's the most up-to-date Steam emu in terms of interfaces and Steam updates, and it's what the team uses by default. You'll see `gbe_fork` referenced in most UC uploads.

## Online-fix.me

Used for multiplayer fixes. The team also maintains in-house alternatives — `uc-online` and `uc-online2` — and sometimes uses Unsteam. Some games work online with `gbe_fork` alone and don't need a separate fix.

## DRM cracks

For titles that need more than a Steam emu, the team usually pulls crack-only files from cs.rin.ru and applies them on top of clean game files.

## Right-click apply script

A GitHub script that adds a right-click menu entry to automatically apply `gbe_fork` to a Steam game folder. It also:

- Generates `interfaces.txt`
- Adds a URL file linking to the site
- Creates a shortcut batch script

Both a Linux and a Windows version are available.

## Accela

Used to download games when CSF isn't available on cs.rin.ru. Thanks to donations to morrenus, the team has **55 daily generations** available on Accela.

## Denuvo

The team does **not** handle Denuvo demo bypasses, and is on the fence about hypervisor-based tools like DenuvOwO because of the risk to end users. Stick to Steamworks-only bypasses.
