# Nothing is working

Some games are stubborn. If the standard workflow doesn't get a title running:

1. Read through the game's cs.rin.ru thread end to end — fixes are often buried in later replies.
2. Confirm you have the latest [`gbe_fork`](/tools-we-use#gbe_fork) build applied.
3. Check whether an Online-fix exists. See [Cracking games](/cracking-games#online-fix).

If none of that works, let the team know. Depending on the title, someone may take a pass at cracking it directly.
