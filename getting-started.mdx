# Getting Started

Getting CSF (Clean Steam Files) - there are mainly 3 sources we use to get game files: ripping directly from Steam using an account which has the game purchased, searching cs.rin forums, or using other platforms that don't include DRM in their games (GOG).

GOG is by far the main one recommended, because no DRM.

## Steam

For this approach you will obviously need a Steam account, with the game you want to download purchased.

We will use Depot Downloader GUI, but if you prefer other depot downloader, feel free to use that one instead.

1. First visit SteamDB, and search up the game you want to download
2. Then copy the number that corresponds to the game's app id, and paste it into DDGUI
3. Go to the depot tab, select the windows depot, copy the corresponding depot id
4. Go to the manifests tab, copy the most recent one
5. In the app enter your steam login details, select where to download the game, and press download

Some games have more than one depot for each platform. For example, Tomb Raider GOTY Edition:

- Ignore all depots not tagged Windows
- Don't download any other languages other than English
- Download all depots: Main content, Executable, English, and DLC depots
- Don't download redists, debug files, or optional depots

Once the files have been downloaded, in each depot delete the .DepotDownloader folder.

At this point, you should have a folder with the steam app id with CSF in it.

## cs.rin

For this approach you will need an account on the forums. You can register by visiting the website.

1. Click the search button or go to the search page
2. Enter the name of the game you want to download and hit enter
3. Find the thread named after your game
4. Go to the last page, work your way back through the thread, and find a download link that works
5. Download the game files

If during any process anything asks for a password, it is most likely cs.rin.ru, unless the uploader specifies otherwise.

Important: make a note of what build id the files are from. Build id indicates version of the game.

## GOG

GOG is a special kind of game store, because the games are DRM-free. GOG should always be checked first.

1. Visit GOG and search for the game you want
2. Download the files
3. After extracting, install the game
4. Go to the install location, and copy all of the game files into a separate folder
5. Take note of the game version written in the name of the setup

GOG includes files in the game folder that aren't needed to run the game. Delete them:
- setup executables
- documentation
- artbooks
- soundtracks

## Cracking the game

This part will only cover simple DRM bypassing. Denuvo, or any other form of DRM than steamworks will not be cracked by us.

This step can be completely skipped if the game is DRM-free.

Apply all cracks/fixes directly. However if an Online fix requires additional dependencies (such as Steam) put them in a _Online-fix folder in the game root.

## Goldberg

Goldberg Steam Emu is a Steam API that can be used as a replacement for the original one to add offline functionality.

Learn how to use it at the Goldberg documentation.

## Steamless

Sometimes just replacing the steam API isn't enough, and running the game still results in an error due to steamstub.

1. Download steamless
2. Run Steamless.exe
3. Click on the three dots, find and select the real game exe
4. Click unpack file
5. Delete the original executable
6. Rename the new one (ending in unpacked.exe) to the name of the original one

## Online-fix

Online-fix is a site that makes cracks for games to allow multiplayer.

1. Make an account on online-fix.me
2. Search the game, click on the correct result
3. Download the fix from the Fix Repair folder
4. Extract it (password: online-fix.me)
5. Put the fix into _Online-fix directory

## Correctly packaging the game

1. Take the root of your game files and put them in a new folder
2. The files should include all DLCs, and no extras like artbooks or soundtracks
3. Put a URL file linking to your site in the folder (rename to UnionCrax.url)
4. Rename the root game folder: {Full game name} - UC
5. Create an archive with 7-Zip
6. Set split to volumes: 10240m (roughly 10 GB)
7. Archive format: .7z
