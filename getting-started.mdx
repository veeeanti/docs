# Getting started

Getting CSF (Clean Steam Files) is the first step in every upload. There are three main sources the team uses:

- Ripping directly from Steam with an account that owns the game
- Searching the cs.rin.ru forums
- Downloading from DRM-free stores such as GOG

GOG is the recommended source whenever it's available because the files ship without DRM.

## Steam

You need a Steam account that owns the game.

The team uses Depot Downloader GUI (DDGUI), but any depot downloader works.

1. Visit SteamDB and search for the game.
2. Copy the app ID and paste it into DDGUI.
3. Open the **Depot** tab, select the Windows depot, and copy the depot ID.
4. Open the **Manifests** tab and copy the most recent manifest.
5. Enter your Steam login details, choose a download location, and press **Download**.

Some titles have multiple depots per platform. For Tomb Raider GOTY Edition, for example:

- Ignore any depot not tagged Windows.
- Skip non-English language depots.
- Download all required depots: main content, executable, English, and DLC.
- Skip redistributables, debug files, and optional depots.

After the download finishes, delete the `.DepotDownloader` folder inside each depot. You should end up with a folder named after the Steam app ID containing the CSF.

## cs.rin.ru

You need a registered account on the cs.rin.ru forums.

1. Open the search page.
2. Search for the game name and press enter.
3. Open the thread that matches the game.
4. Jump to the last page and work backward until you find a working download link.
5. Download the game files.

If anything asks for a password during extraction, it is almost always `cs.rin.ru` unless the uploader specifies otherwise.

<Tip>
Make a note of the **build ID** of the files. Build ID indicates the game version and is required when you list the game later.
</Tip>

## GOG

GOG should always be checked first because the games are DRM-free.

1. Visit GOG and search for the game.
2. Download the installer.
3. Run the installer locally.
4. Copy the installed game files into a separate folder.
5. Note the game version from the setup file name.

GOG installers include extras that the upload should not contain. Delete:

- Setup executables
- Documentation
- Artbooks
- Soundtracks
