# Other tools we have to use

I have a script on my github that you can add to your right-click menu to automatically apply gbe_fork to steam games, interfaces.txt is also generated automatically as well as a url file linking to our site. And also the shortcut batch script that will create a shortcut to the game executable on your desktop that can have all launch args preset to make it easy for users to keep their shortcuts working even after they move the game folder by accident and break it, just run it again to fix the paths! The script I made has both a linux version and also the standard windows version.

Our file mirrorer is broken currently.
