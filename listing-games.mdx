# Listing Games

After you are done uploading your game with our tool, we need to list it.

1. Copy the output json of the tool
2. Fill the fields in the game management page
3. Include the version as labeled on the game upload
4. Include the source of your crack and size of the unarchived game folder

Crack source means that if you used goldberg, say you used goldberg. If you used GOG instead of Steam say you used GOG. If you got it from a scene release or a repacker, then you check what scene group cracked the game, and list them.

Then click add/update game. Our tool also has a link to game (auto push links) feature, so you don't have to manually copy over the output json.

Remember, this guide might be slightly outdated, so if you have any questions don't hesitate to ask.
