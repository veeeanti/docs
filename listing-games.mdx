# Listing games

After you finish uploading a game with the team tool, list it on the site.

1. Copy the output JSON from the upload tool.
2. Open the game management page and fill in the fields.
3. Set the version to match the label on the upload (build ID for Steam, version string for GOG).
4. Set the crack source and the unarchived game folder size.
5. Click **Add/Update game**.

**Crack source** means whichever bypass you used. If you used Goldberg, say Goldberg. If you used GOG instead of Steam, say GOG. If the files came from a scene release or a repacker, list the responsible scene group.

<Tip>
The upload tool also has an auto-push link feature, so you usually don't need to manually copy the output JSON.
</Tip>
