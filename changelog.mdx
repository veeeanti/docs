## Week of May 16, 2026

### New features

- **veeWiki is live.** The official UnionCrax team wiki launched this week, covering the full workflow from sourcing to listing. Start at [Getting started](/getting-started).
- **Acquisition guide.** End-to-end guide for pulling Clean Steam Files from Steam (via Depot Downloader), GOG, and cs.rin.ru, with per-source steps and gotchas. See [Downloading games](/downloading-games).
- **Tooling reference.** Single page documenting the team toolchain — `gbe_fork`, Online-fix.me, Accela, Steamless, and the right-click apply script (now with both Windows and Linux versions). See [Tools we use](/tools-we-use).
- **Packing standard.** Published the UC-style packing convention: 7-Zip, `- UC` folder suffix, 10 GB split volumes. See [Packing games](/packing-games).
- **Listing flow.** Documented how to list a game after upload, including the auto-push links feature in the internal tool. See [Listing games](/listing-games).
- **ROMHEAVEN added as a source** for ROMs and CSF alongside the existing options. See [Downloading games](/downloading-games).

### Updates

- **Accela quota raised to 55 daily generations** thanks to donations to morrenus. See [Tools we use](/tools-we-use).
- **Multi-part uploads no longer required.** You can upload single archives; the 10 GB split is now a recommendation, not a rule. See [Packing games](/packing-games).
- **Cracking guidance refreshed.** Clarified that the team handles Steamworks bypasses only — not Denuvo — and when to reach for Steamless if an emu alone isn't enough. See [Cracking games](/cracking-games).
- **In-house multiplayer fixes** (`uc-online` and `uc-online2`) are now documented as alternatives to Online-fix.me. See [Tools we use](/tools-we-use).
- **Troubleshooting page added** with a short path for when nothing works on a given title. See [Nothing working](/nothing-working).

### Known issues

- **File mirrorer is currently broken.** A fix is in progress. See [Tools we use](/tools-we-use).
