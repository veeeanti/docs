## Week of May 16, 2026

### New features

- **veeWiki is live.** The official UnionCrax team wiki launched this week. It covers the full workflow for sourcing, preparing, and listing games. Start at [Getting started](/getting-started).
- **Acquisition guide.** New end-to-end guide for pulling Clean Steam Files from Steam (via Depot Downloader), GOG, and cs.rin.ru. See [Getting started](/getting-started) and [Downloading games](/downloading-games).
- **Tooling reference.** A single page documenting the toolchain the team uses — `gbe_fork`, Online-fix.me, Accela, Steamless, and the right-click apply script. See [Tools we use](/tools-we-use).
- **Packing standard.** Published the current UC-style packing convention: 7-zip, `- UC` suffix on the folder, 10 GB split volumes. No more required multi-part uploads. See [Packing games](/packing-games).
- **Listing flow.** Documented how to list a game after upload, including auto-push links from the internal tool. See [Listing games](/listing-games).

### Updates

- **Accela quota bumped to 55 daily generations** thanks to donations to morrenus. Details in [Tools we use](/tools-we-use).
- **Cracking guidance refreshed.** Clarified which DRM the team handles (Steamworks bypass only — not Denuvo) and how to use Steamless when an emu alone isn't enough. See [Cracking games](/cracking-games).
- **Troubleshooting page.** Added a short path for when nothing works on a given title. See [Nothing working](/nothing-working).
